# Math-Helper

The goal of this program is to take input from the user and do the math work for them. Not only by providing them with the answer, but also providing them with the step-by-step. Along with that, the finished program will have a GUI and different branches of math (Algebra, Discrete, Calculus, etc.). For now, all actions will be done in the terminal.
